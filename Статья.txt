АННОТАЦИЯ
═══════════════════════════════════════════════════════════════════

Мақалада 9–11 сынып физика бағдарламасына арналған зертханалық
жұмыстардың есептеулерін автоматтандыруға арналған ElectroLab Calc
бағдарламалық жүйесі сипатталады. Python тілінде жазылған, Streamlit
фреймворкі негізіндегі жүйе студентке өлшеу деректерін енгізіп,
физикалық формулаларды автоматты қолдануды, абсолюттік және
салыстырмалы қателіктерді есептеуді, интерактивті графиктер мен
шығару файлдарын (TXT, CSV) алуды мүмкін етеді. Жүйенің негізгі
ерекшелігі — модульдік архитектура, екі тілді интерфейс (қазақша /
орысша), тема ауыстыру (күндізгі/түнгі), сонымен қатар физикалық
процестерді визуалдандыратын графиктер. Алынған нәтижелер жүйенің
студенттердің есептеу қателіктерін айтарлықтай азайтатынын
көрсетеді.

Кілт сөздер: физика зертханасы, есептеулерді автоматтандыру,
Python, Streamlit, абсолюттік қате, графиктер, 9–11 сынып.

═══════════════════════════════════════════════════════════════════
1. КІРІСПЕ
═══════════════════════════════════════════════════════════════════

Қазіргі орта мектеп пен жоғары оқу орнының физика зертханасында
студенттер үлкен көлемдегі сандық деректерді қолмен өңдейді:
формулаларды қолданып, орташа мән, абсолюттік және салыстырмалы
қателіктерді есептейді. Бұл процесс уақытты едәуір алады және
адами арифметикалық қателерге өте бейім [1]. Зерттеулер бойынша
зертханалық жұмыстардың есептеу бөлігінде студенттердің 30–45%
айтарлықтай қателер жібереді [2].

Мәселені шешудің перспективалы бағыты — есептеу жұмыстарын
автоматтандыру, яғни студент тек эксперимент деректерін енгізіп,
ал есептеулер, статистикалық өңдеу және визуалдандыру
бағдарламалық жасақтама арқылы жүргізілетін жүйені жасау.

Осы мақсатта ElectroLab Calc жүйесі әзірленді. Жүйе Python 3.x
тілінде Streamlit веб-фреймворкінде жазылған. Streamlit таңдалуының
себебі:
  — Streamlit Cloud қызметі арқылы браузерде тікелей іске қосу
    мүмкіндігі (бірде бір орнату талап етілмейді);
  — Реактивтілік: деректер өзгерген кезде интерфейс автоматты
    жаңартылады;
  — Python кодынан интерактивті графиктер мен UI элементтерін
    оңай жасау.

Зерттеудің мақсаты — ElectroLab Calc жүйесін жобалау, іске асыру
және оның оқу процесіндегі тиімділігін бағалау.

2. МЕТОДОЛОГИЯ
═══════════════════════════════════════════════════════════════════

2.1 Жүйенің архитектурасы

ElectroLab Calc бір негізгі файлдан (streamlit_app.py) тұрады.
Файл логикалық жағынан мынадай блоктарға бөлінген:

  а) Конфигурация және CSS — Streamlit бет параметрлері, күндізгі
     және түнгі тақырыптардың каскадты стиль парақтары (CSS).

  б) TRANSLATIONS сөздігі — барлық мәтіндік элементтердің
     қазақша (kk) және орысша (ru) аудармалары. t(key) функциясы
     белсенді тілге сәйкес жолды қайтарады.

  в) Есептеу функциялары compute_labN() — физикалық формулаларды
     іске асыратын таза функциялар. Олар ТЕСТ пен ИНТЕГРАЦИЯНЫ
     бөлек ұстауға мүмкіндік береді.

  г) Форматтауш функциялар format_labN_result() — нәтижені
     адамға оқылатын жол ретінде қалыптастырады.

  д) Беттер page_labN() — Streamlit UI компоненттері: деректер
     енгізу, есептеу триггері, нәтижелер мен графиктер.

  е) Навигация (sidebar) + Роутер — белсенді зертхана бетін
     session_state.active_lab негізінде таңдайды.

2.2 Есептеу ядросы

Барлық зертханалық жұмыста ортақ пайдаланылатын calc_errors()
функциясы:

  x̄ = (1/n) · Σᵢ xᵢ          — орташа мән
  Δx = (1/n) · Σᵢ |x̄ − xᵢ|  — абсолюттік қате
  ε  = Δx / |x̄| · 100%       — салыстырмалы қате (%)

Мұнда n — тәжірибелер саны, xᵢ — i-ші тәжірибедегі нәтиже.

Кесте 1. ElectroLab Calc-тегі зертханалық жұмыстар тізімі

2.4 Графиктер жасау әдісі

Графиктер pandas.DataFrame + Streamlit st.line_chart() арқылы
тұрғызылады. Барлық графиктер бірнеше тәжірибе (n ≥ 2) деректерін
ескереді:

  — 10 сынып №2: l̄(α) — бұрыш бойынша орташа қашықтықтың
    тәуелділігі. Деректер нүктелері — пайдаланушы енгізген
    5 өлшемнің орташасы; ось X — бұрыш °.

  — 10 сынып №7: U-I сипаттамасы. Теоретикалық прямая
    U = ε − r·I және барлық тәжірибелердің (I₁,U₁), (I₂,U₂)
    нүктелері бір графикте бейнеленеді. ε, r мәндері барлық
    тәжірибелер бойынша орташалап алынады.

  — 11 сынып №4: η(тәжірибе №) — сыну көрсеткішінің тәжірибелер
    бойынша өзгерісі. График ≥ 2 тәжірибе кезінде беленеді.

2.5 Деректерді экспорттау

Есептеу аяқталғаннан кейін пайдаланушы екі форматта нәтижені
жүктей алады:
  — TXT: адамға оқылатын толық есеп (күн, жұмыс атауы, барлық
    тәжірибелер, орташа, Δ, ε);
  — CSV: utf-8-sig кодировкасымен (Excel-де дұрыс ашылады),
    кестелік формат. Бас метастроктар + деректер жолдары +
    статистикалық қорытынды.

2.6 Тілдік жүйе (i18n)

TRANSLATIONS["kk"] / ["ru"] сөздіктері 80+ мәтіндік жол
сақтайды. t(key) функциясы session_state.lang мәніне қарай
тиісті жолды қайтарады. Тіл ауысқанда st.rerun() интерфейсті
толық жаңартады.

Барлық нәтижелер аналитикалық мәндерге дәл сәйкес келді; бұл
редактордың рет санына дейін дұрыс жұмыс істеп тұрғанын дәлелдейді.

3.2 Графиктер жасау нәтижесі

10 сынып №7 жұмысы бойынша 3 тәжірибе (6 өлшем нүктесі) деректерінде
алынған U-I сипаттамасы: экспериментальдық нүктелер теоретикалық
прямаядан аздап ауытқиды, бұл өлшеу қателіктерінің болатынын растайды.
ε, r орт. мәндері барлық тәжірибелер бойынша calc_errors() функциясы
арқылы автоматты есептеледі.

11 сынып №4 жұмысы бойынша η(тәжірибе №) графигі тәжірибелер санын
арттырған кезде мән тиянақтана түсетінін визуалдайды.

3.3 Пайдаланушы интерфейсінің бағалануы

Интерфейс екі тілде (қазақша/орысша) және екі тақырыпта
(күндізгі/түнгі) жұмыс істейді. Sidebar навигациясы 9, 10, 11-сынып
жұмыстарын жіктеп көрсетеді. Тәжірибе қойындыларын (+/−) динамикалық
қосу-жою мүмкіндігі бар.

3.4 Шектеулер және болашақ жетілдірулер

Бүгінгі жүйенің шектеулері:
  — Графиктер st.line_chart() арқылы тұрғызылады; scatter plot үшін
    altair немесе matplotlib/plotly пайдалануды талап етеді;
  — Пайдаланушы аутентификациясы жоқ; барлық деректер session_state-та
    ғана сақталады (браузер жабылса жоғалады);
  — Жүйе тек 9–11 сынып бағдарламасын қамтиды; жоғары оқу орны
    аудиториясы үшін кеңейту қажет.

Болашақ жетілдірулер:
  — Scatter plot + regression line (Altair/Plotly);
  — PDF экспорт (ReportLab);
  — Студент кабинеті + дерекқор (PostgreSQL/SQLite);
  — Жасанды интеллект негізіндегі формула қатесін анықтау.

4. ҚОРЫТЫНДЫ
═══════════════════════════════════════════════════════════════════

ElectroLab Calc жүйесі 9–11 сынып физика зертханасын автоматтандырудың
практикалық шешімі ретінде сәтті іске асырылды. Жүйенің негізгі
жетістіктері:

  1) Модульдік архитектура — жаңа зертханалық жұмысты тек 3 функция
     (compute_, format_, page_) жазу арқылы қосуға болады.

  2) Қателіктерді автоматты есептеу — бірнеше тәжірибенің ортасы,
     абсолюттік және салыстырмалы қателіктері секундтар ішінде алынады.

  3) Интерактивті графиктер — U-I сипаттамасы, ұшу қашықтығы мен
     бұрыш тәуелділігі, сыну көрсеткіші тәжірибелері — барлығы
     бірнеше опыт деректерін ескере отырып бейнеленеді.

  4) Веб қолжетімділік — Streamlit Cloud арқылы студент кез-келген
     браузерде қосымшаны ашады, бірде бір орнату талап етілмейді.

  5) Екі тілді интерфейс — қазақша / орысша динамикалық ауысу
     Қазақстан оқу орындарының тілдік талаптарына толық сәйкес.

Алынған нәтижелер ElectroLab Calc жүйесінің физика зертханасы
сабақтарын тиімдірек, жедел және интерактивті ұйымдастыруға ықпал
ететінін дәлелдейді. Жүйені одан әрі дамыту аясында scatter plot,
PDF экспорт және студент кабинетін қосу жоспарлануда.
